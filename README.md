# DebugPadawan's AntiGravity Skills

A collection of specialized skills designed for enhancing developer performance and maintaining high code standards with the AntiGravity AI assistant.

## Available Skills

### 1. [Context Minimizer](./context-minimizer)
**Description**: Forces extreme conciseness by eliminating conversational filler and restricting code output to minimal diffs. 
**Usage**: When you want to save context window space and reduce noise.

### 2. [Dependency Checker](./dependency-checker)
**Description**: Analyzes the impact of code modifications on dependent functions.
**Usage**: When modifying, refactoring, or deleting existing code to prevent side-effects across the codebase.

### 3. [Unique Design Enforcer](./unique-design-enforcer)
**Description**: Forces unique, non-generic, and opinionated UI/UX designs.
**Usage**: When generating HTML/CSS, UI components, or website layouts to avoid the predictable "AI aesthetic".

---

## How to use these skills

Add these skill folders to your AntiGravity `skills` directory (usually located at `.gemini/antigravity/skills` in your workspace) to enable them.
Each folder contains a `SKILL.md` where the logic and instructions for that skill are defined.
